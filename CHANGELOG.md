# Changelog

## Unreleased

## 1.5.8 - 2026-06-26

+ [修复] IMAP 调度台改为账号登录表单维护，不再点击添加时生成多个配置框，也不提供 IMAP 文本导入。

## 1.5.7 - 2026-06-26

+ [修复] 注册取码改为本机 IMAP 调度台轮播账号池，不再运行时调用外部取码接口。
+ [新增] 顶部新增“调度台”入口，可维护 IMAP 账号池并供注册/补 login 统一取码。

## 1.5.6 - 2026-06-26

+ [修复] 旧 iOS 兼容构建改为选择性降级可选链、空 catch 和空值合并语法，避免全量转译破坏 Next runtime。

## 1.5.5 - 2026-06-25

+ [修复] Web 构建后将静态 JS 转译到 iOS 11 兼容语法，避免旧版 iOS Safari 因可选链等现代语法白屏。
+ [修复] 登录页提前注入旧 iOS polyfill，并将内联主题脚本与旧 chunk 恢复脚本改为旧 Safari 可解析写法。

## 1.5.4 - 2026-06-25

+ [修复] 首页改为无 JS 也可见并自动进入登录页，避免手机端客户端跳转失败时白屏。
+ [修复] 旧静态 JS chunk 缺失时返回自动刷新脚本，帮助手机浏览器从旧缓存恢复到新版页面。

## 1.5.3 - 2026-06-25

+ [修复] Web 首页 HTML 禁止缓存，避免升级后手机浏览器继续加载旧静态 chunk。

## 1.5.2 - 2026-06-25

+ [修复] Docker 镜像启动时直接使用已构建虚拟环境，避免运行时联网同步 dev 依赖。

## 1.5.1 - 2026-06-25

+ [新增] 注册页支持手动导入邮箱进入待注册队列，注册取码统一走邮箱调度台。
+ [新增] 新增恢复登录/补 login 标签页，支持导入邮箱后独立补登。
+ [新增] 新增 mimo 代理配置页，注册线程按本地 listener 绑定不同代理出口。
+ [优化] 注册任务新增队列、失败池和调度台配置持久化。

## 1.5.0 - 2026-06-13

+ [新增] 新增 WARP / Privoxy / FlareSolverr 清障方案，注册遇到 Cloudflare 拦截后可刷新 clearance 并重试。
+ [新增] 新增 `outlook_token` 邮箱池，支持 Outlook/Hotmail 注册验证码读取。
+ [新增] 新增网页搜索兼容接口、图片编辑 mask 和图片任务相关能力。
+ [优化] 更新 sentinel/PoW 获取方式，提高上游请求兼容性。
+ [优化] 调整代理优先级和注册请求重试逻辑。

## 1.4.1 - 2026-06-03

+ [新增] 账号刷新改为异步模式，支持前端轮询刷新/重新登录进度。
+ [新增] 号池管理页面新增重新登录功能，支持密码登录恢复异常账号。
+ [新增] 刷新后自动重新登录异常账号（可在设置页开启）。
+ [新增] 图片生成支持并行模式，多张图片使用独立线程和账号同时生成。
+ [新增] 图片轮询超时自动换账号重试（最多4次），连接超时同账号递增等待重试。
+ [新增] 图片二次确认机制与先check再hit可配置化，关闭后可跳过等待直接返回结果。
+ [新增] 图片任务进度追踪，显示当前生成步骤（上传/预热/获取token/生成中等）。
+ [新增] 图片超时后续轮询功能，前端显示"继续等待"按钮。
+ [新增] 设置页新增图片二次确认、超时等待时间、自动重新登录等配置项。
+ [优化] 优化生图页面滚动加载性能，图片懒加载、会话切换滚动位置保存与恢复。
